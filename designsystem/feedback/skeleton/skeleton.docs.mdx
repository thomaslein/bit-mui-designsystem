import { Stack } from '@kreftregisteret/designsystem.layout.stack';
import { Box } from '@kreftregisteret/designsystem.layout.box';
import { Skeleton } from './skeleton.js';

The data for your components might not be immediately available. You can improve the perceived responsiveness of the page by using skeletons.
It feels like things are happening immediately, then the information is incrementally displayed on the screen

## Basic usage

The component is designed to be used **directly in your components**. For instance:

```js
import { Skeleton } from '@kreftregisteret/designsystem.feedback.skeleton';

const height = 118;
const width = 210;

// ...

{
  item ? (
    <img
      style={{
        width,
        height,
      }}
      alt={item.title}
      src={item.src}
    />
  ) : (
    <Skeleton variant="rectangular" width={width} height={height} />
  );
}
```

## Skeleton shapes

The component supports 4 shape variants:

- `text` (default): represents a single line of text (you can adjust the height via font size).
- `circular`, `rectangular`, and `rounded`: come with different border radius to let you take control of the size.

```js live
() => (
  <Stack spacing={1}>
    <Skeleton variant="text" sx={{ fontSize: '1rem' }} />
    <Skeleton variant="circular" width={40} height={40} />
    <Skeleton variant="rectangular" width={210} height={60} />
    <Skeleton variant="rounded" width={210} height={60} />
  </Stack>
);
```

## Skeleton animation

By default, the skeleton pulsates, but you can change the animation to a wave or disable it entirely.

```js live
() => (
  <Box sx={{ width: 300 }}>
    <Skeleton />
    <Skeleton animation="wave" />
    <Skeleton animation={false} />
  </Box>
);
```
