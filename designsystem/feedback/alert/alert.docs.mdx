Alerts give users brief and potentially time-sensitive information in an unobtrusive manner and require the user's acknowledgment.

## Basic usage

The Material UI Alert component includes several props for quickly customizing its styles to provide immediate visual cues about its contents.

```js live
() => (
  <Alert icon={<CheckIcon fontSize="inherit" />} severity="success">
    Here is a gentle confirmation that your action was successful.
  </Alert>
);
```

## Alert severity

The severity prop accepts four values representing different states: `success` (the default), `info`, `warning`, and `error` with corresponding icon and color combinations for each:

```js live
() => (
  <Stack sx={{ width: '100%' }} spacing={2}>
    <Alert severity="success">This is a success Alert.</Alert>
    <Alert severity="info">This is an info Alert.</Alert>
    <Alert severity="warning">This is a warning Alert.</Alert>
    <Alert severity="error">This is an error Alert.</Alert>
  </Stack>
);
```

## Description alert

The Alert component can includes the `AlertTitle` component to display a title for the alert:

```js live
() => (
  <Alert severity="info">
    <AlertTitle>Info</AlertTitle>
    This is an info alert with a title.
  </Alert>
);
```
