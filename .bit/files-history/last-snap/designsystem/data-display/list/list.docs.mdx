Lists are continuous, vertical indexes of text or images. They are composed of items containing primary and supplemental actions, which are represented by icons and text.

## Basic usage

The List component is a continuous vertical index of text or images.
A List is composed of `ListItem` components (also provided by this component).

```js
import {
  List,
  ListItem,
  ListItemButton,
  ListItemText,
  ListItemIcon,
  ListSubheader,
  ListItemAvatar,
} from '@kreftregisteret/designsystem.data-display.list';

() => (
  <List>
    <ListItem>
      <ListItemButton>
        <ListItemIcon>
          <InboxIcon />
        </ListItemIcon>
        <ListItemText primary="Inbox" />
      </ListItemButton>
    </ListItem>
  </List>
);
```
