---
description: Progress indicators commonly known as spinners, express an unspecified wait time or display the length of a process
---

import { Stack } from '@kreftregisteret/designsystem.layout.stack';
import { CircularProgress } from './circular-progress.js';
import { LinearProgress } from './linear-progress.js';

Progress indicators inform users about the status of ongoing processes, such as loading an app, submitting a form, or saving updates.

- **Determinate** indicators display how long an operation will take.
- **Indeterminate** indicators visualize an unspecified wait time.

The animations of the components rely on CSS as much as possible to work even before the JavaScript is loaded.

## Basic usage

Use the `CircularProgress` or `LinearProgress` components to create determinate and indeterminate progress indicators.

```js
import {
  CircularProgress,
  LinearProgress,
} from '@kreftregisteret/designsystem.feedback.progress';

() => (
  <>
    <CircularProgress />
    <LinearProgress />
  </>
);
```

## Circular indeterminate and determinate progress indicator

```js live
() => (
  <Stack spacing={2}>
    <CircularProgress />
    <CircularProgress variant="determinate" value={25} />
  </Stack>
);
```

## Linear indeterminate and determinate progress indicator

```js live
() => (
  <Stack sx={{ width: 400 }} spacing={2}>
    <LinearProgress />
    <LinearProgress variant="determinate" value={25} />
  </Stack>
);
```

## Progress indicator colors

```js live
() => (
  <Stack sx={{ width: 400 }} spacing={2}>
    <LinearProgress color="secondary" />
    <LinearProgress color="success" />
    <LinearProgress color="inherit" />
  </Stack>
);
```
